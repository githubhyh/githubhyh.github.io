VLOOKUP函数

语法

  • lookup_value: 要查询的值
  • table_array: 查询范围（二维表格范围）
  • col_index_num: 返回查询范围中的哪一列，二维表格中第一行的列信息，可能有多列，需要指定返回第几列，一般返回第一列，参数值则为：1
  • range_lookup: 可选参数，是否模糊匹配，false表示精确匹配。默认true

注意事项

  • 大小写敏感: VLOOKUP函数是大小写不敏感的。大小写敏感匹配可以使用index和match函数组合
  • 数据排序: 如果模糊匹配，则查询范围第一列的数据必须按升序排列

使用示例

  • 说明：采用精确查询，在B列中查询A列的值，返回B列的值，如果没有则返回#N/A

INDEX 和 MATCH

INDEX 语法

  • array: 查询的区域，某一列或某一行范围
  • row_num: 要返回的行号
  • column_num: 可选，要返回的列号。array为单行或单列是可省略

INDEX 返回值

返回array中指定位置的值，如果不存在，则会报error，可以用IFERROR处理

MATCH 语法

  • lookup_value: 要查询的值
  • lookup_array: 查询范围，某一列或某一行
  • match_type: 可选，匹配类型
    • 1：查找小于或等于 lookup_value 的最大值（数组必须按升序排列）
    • 0：查找等于 lookup_value 的第一个值（精确匹配）
    • -1：查找大于或等于 lookup_value 的最小值（数组必须按降序排列）

MATCH 返回值

返回一个数字，表示查询值在array中的位置，如果没有查询到，则返回#N/A
